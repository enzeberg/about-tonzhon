铜钟 (Tonzhon): 音乐网站

铜钟是一个主打 [听歌] 功能的 web app, 致力于为人们带来卓越的听歌体验。铜钟有着丰富的音乐资源，简洁清爽的 UI 和方便的交互。在铜钟上，你不仅可以方便地找到并聆听你喜欢的歌曲，还可以将它们保存下来。

铜钟上的一切内容都是与音乐直接相关的，没有直播, 社交和广告，不会干扰你的听歌心情。在铜钟上，你可以沉浸到属于你自己一个人的那片天地，忘却世间的纷纷扰扰...

铜钟的上线时间

铜钟上线于 2017 年秋季，那时用的域名是 tongzhong.xyz。2021 年春季，铜钟的域名被更换为了 tonzhon.com。

铜钟的使命

  • 为人们带来卓越的听歌体验, 丰富人们的生活.
  • 帮助人们在学习和工作时提神醒脑, 保持专注, 忘却烦恼, 改善心情, 净化心灵, 激发创造力...
  • 促进社会生产力.

铜钟的使用

铜钟推荐的浏览器

推荐在 Microsoft Edge, Google Chrome 中使用铜钟 (铜钟在被开发和测试时, 用的浏览器是 Microsoft Edge).

关于铜钟账户

  • 推荐使用 Microsoft 登入, 因为使用 Microsoft 登入时无需在铜钟上设置用户名和密码, 你只需要记住你的 Microsoft 账户即可.
  • 使用 用户名/密码 登入时, 推荐使用浏览器记住你的用户名和密码, 因为忘记密码后会比较麻烦.
  • 铜钟用户名具有唯一性, 注册时不区分大小写 (如 "Enze" 和 "enze" 会被视为同一个用户名), 但在登入时区分大小写.
  • 铜钟没有 [密码找回] 功能, 因为你的密码已被铜钟加密, 铜钟并不知道你的密码.
  • 铜钟账户是铜钟的宝贵资源, 请勿占着不用, 更不要恶意占用铜钟的用户名资源.

for 手机

手机关闭屏幕后可能导致 Tonzhon.com 无法自动播放, 所以在用手机使用 Tonzhon.com 时, 请不要让手机关闭屏幕.

for iPhone

iPhone 对浏览器播放音频有限制, 所以在 iPhone 上使用 Tonzhon.com 不如在其他手机上方便.

for tablets (eg: iPad, Surface Pro, 华为 MatePad, Xiaomi Pad...)

建议使用铜钟的桌面版本 (需要将你的浏览器设置为 "请求桌面网站").

隐藏操作

  • 双击歌曲可播放

快捷键

  • 空格键: 播放 / 暂停

Q&A

为何起名为 "铜钟"?

因为铜钟是我们中国古代的一种乐器，它与音乐有着很强的相关性。人们看到 "铜钟" 二字会联想到音乐。

为何铜钟上面没有广告?

因为广告会影响用户体验。铜钟是一个极其重视用户体验的网站，铜钟不希望用户受到广告的干扰。一旦铜钟上出现了明显的广告，那铜钟就不酷了。

为何铜钟没有 评论/回复 等社交功能?

因为社交会干扰人们的听歌心情，例如，对于周杰伦的《最伟大的作品》，有些人觉得很好听，有些人却觉得不好听，觉得好听的人在看到有人说它不好时是会难过的。铜钟认为每一个人都是一个高度独立的个体，铜钟不希望人们听歌时受到他人的干扰。

所以，铜钟不想引导用户进行社交，更不会加入 [直播] 这样的社交功能。

为何铜钟的英文名字是 "Tonzhon"?

起初铜钟的英文名字是 "Tongzhong", 后来觉得太长，遂改成了 "Tonzhon". 在众多中文输入法中，输入 "tonzhon" 是可以打出 "铜钟" 的。

为何铜钟的 header 上面只有 "Artists" 字样, 而没有 "歌手" 或 "艺人" 字样?

举些例子吧, 周杰伦是一个歌手，林海是一个艺人 (音乐人, 作曲家), 五月天是一个组合 (乐队), 但 Two Steps from Hell 既不能算作一个歌手，又不能算作一个艺人，甚至不能算作一个组合，因为严格来说 Two Steps from Hell 是一家音乐制作公司. 而 "artist" 一词比较宽泛, 可以把 "歌手", "艺人", "组合/乐队", "音乐制作公司" 等意包含在内.

中文词汇往往比较具体, 比如以 "人", "手", "家" 等字结尾的词只能指代个体化的人, 而无法指代 由多人组成的团体 或 非人的实体. 英文更加注重逻辑, 词汇的意思比较宽泛, 以 "er" 或 "or" 结尾的词可以指代中文里的以 "人", "者", "家", "手", "器" 等字结尾的词.
